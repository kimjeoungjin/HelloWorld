#define __HAL_RCC_GET_SMPS_ADC_ANA_DIV_CONFIG ( )    LL_RCC_GetSMPS_ANADIV_Prescaler()

Macro to get the SMPS ADC ANA clock DIV prescaler.

Return values
SMPSADC ANA div prescaler. This parameter can be one of the following values:
  • RCC_SMPS_ADC_ANADIV1 SMPS and ADC clock division 1 (ADC/SMPS source clock is 32 MHz)
  • RCC_SMPS_ADC_ANADIV2 SMPS and ADC clock division 2 (ADC/SMPS source clock is 16 MHz)
  • RCC_SMPS_ADC_ANADIV4 SMPS and ADC clock division 4 (ADC/SMPS source clock is 8 MHz)

#define __HAL_RCC_LSCO_CONFIG (   __LSCOCLKSOURCE__)    LL_RCC_ConfigLSCO((__LSCOCLKSOURCE__))

Macro to configure the LSCO clock.

Parameters
<strong>LSCOCLKSOURCE</strong>specifies the LSCO clock source. This parameter can be one of the following values:
  • RCC_LSCOSOURCE_NOCLOCK LSCO output disabled
  • RCC_LSCOSOURCE_LSI_LPMU Internal LPMU slow clock source as LSCO source
  • RCC_LSCOSOURCE_LSI LSI clock selected as LSCO source
  • RCC_LSCOSOURCE_LSE LSE clock selected as LSCO sourcee
